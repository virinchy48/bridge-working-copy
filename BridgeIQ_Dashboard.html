Applicable Standards & Reference Documents
AS 5100
Bridge Design Standard
SAA/SNZ Australian/New Zealand Standard for bridge design. Covers loads (M1600, SM1600), structural analysis, seismic, fatigue, and inspection cycles. Parts 1–9.
Design LoadInspectionCondition Rating
BIMM
Bridge Inspection & Maintenance Manual
Austroads publication AP-R678. Standard methodology for principal, routine and general bridge inspections. Defect categories, severity ratings, and BCI computation.
InspectionDefectBCI
BANC Ed.4
Bridge & Asset Network Condition
Austroads national data exchange format for bridge and culvert condition data. CSV export schema: STRUCTURE, INSPECTION, DEFECT tables. Submitted to state road agencies via data portal.
Data ExchangeCSVState Reporting
TfNSW TS01501
Asset Classification System v3.01
Technical Standard defining TOC codes (BR-BRDG, DG-CULV), bridge type-of codes, value lists and SAP integration requirements. Maps to BRIDGE_INFRA class characteristics.
SAP IntegrationTOC CodesEQUI Chars
NHVR HVSM
Heavy Vehicle Safety & Mass Standards
National Heavy Vehicle Regulator standards for mass limits (GML, CML, HML), dimension limits, and NHVR access conditions. Drives restriction posting and permit requirements.
Mass LimitsPermitsRestrictions
GDA2020
Geocentric Datum of Australia 2020
Australian geodetic datum (EPSG:7843). All NHVR bridge coordinates stored and exported using GDA2020. ESRI ArcGIS feature service configured with this spatial reference.
CRSESRIEPSG:7843
SAP S/4HANA 2023
Plant Maintenance + Asset Accounting
SAP PM module used for Functional Location hierarchy (IFLOT), Equipment Master (EQUI), Work Orders (AUFNR), and Notifications. FI-AA integration for capitalised asset tracking.
SAP PMEQUIFI-AA
OData V4
SAP CAP Open Data Protocol V4
REST-based API protocol used for all NHVR backend services. Supports $filter, $expand, $orderby, bound actions, and function imports. JSON response format. JWT auth via XSUAA.
